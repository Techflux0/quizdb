Try it

GET https://quizdb-725t.onrender.com/quizes

Get all quizzes from a specific collection

Parameters:

  • collectionName - Name of the quiz collection (e.g. "science")

Example Response:

{
  "success": true,
  "collection": "science",
  "count": 10,
  "data": [
    {
      "question": "What is the chemical symbol for gold?",
      "options": ["Au", "Ag", "Pb", "Fe"],
      "answer": "Au"
    }
    // ... more questions
  ]
}

GET https://quizdb-725t.onrender.com/quizes/

Search quizzes within a collection

Parameters:

  • collectionName - Name of the quiz collection
  • search - Search term to filter questions

Example Response:

{
  "success": true,
  "collection": "science",
  "count": 2,
  "data": [
    {
      "question": "What is the chemical symbol for gold?",
      "options": ["Au", "Ag", "Pb", "Fe"],
      "answer": "Au"
    }
  ]
}
